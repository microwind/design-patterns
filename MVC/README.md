# MVC 分层架构设计概述

模型-视图-控制器（Model-View-Controller，简称 MVC）是一种经典软件架构模式，通过职责分离实现代码的可维护性和可扩展性。MVC 适用于需要清晰分离用户界面、业务逻辑和数据管理的应用场景。

## MVC 结构图形示例
以Web后端开发为例
```text
                     用户请求  
                       |  
                       v
+---------+       +-----------+      +-----------+
|  View   |  <--- | Controller| ---> |   Model   |
+---------+       +-----------+      +-----------+
    ^                                      v
    |            Model数据映射到View         |
    ****--------------------------------****
```
