# 应用实践例子

本目录通过实际例子来说明设计模式的用途。
   
## 目录说明：
1. springwind

  springwind是一个类似Spring MVC的极简原生实现，不依赖Spring MVC框架。核心功能涵盖基本的IoC（控制反转）容器、Web请求转发及数据库操作等，通过该框架，开发者可深入剖析并理解Web开发的核心原理。通过这个实例，将有助于您彻底了解Spring的运作机制。

2. knife

knife是一个基于Spring Boot 3.x 的DDD工程脚手架，严格遵循DDD（领域驱动设计）模式搭建。脚手架内置数据库连接、Web安全等核心模块的实践实例，借助这些实例，开发者可直观掌握Web开发全流程。本脚手架能够助力开发者快速构建高质量代码，提升项目开发效率。
